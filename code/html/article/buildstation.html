要搭建一个网站，并让别人能够访问你的网站，就需要一台服务器
  • 申请服务器
    • 服务器申请：本人的申请的是阿里云服务器，所以直接到阿里云上买就行了，选用的系统是ubuntu 16.04.6 LTS
  • 安装web服务器软件
    • web服务器有很多种，我选用的是ngnix,安装环境linux ubuntu 16.04.6 LTS,下面是安装步骤
    1. 安装必要依赖
      2. sudo apt install curl gnupg2 ca-certificates lsb-release
    2. 设置ngnix包的仓库
      3. echo "deb http://nginx.org/packages/mainline/ubuntu `lsb_release -cs` nginx" \ | sudo tee /etc/apt/sources.list.d/nginx.list
    3. 导入ngnix官方签名验证真假
      4. curl -fsSL https://nginx.org/keys/nginx_signing.key | sudo apt-key add -
    4. 验证真假
      5. sudo apt-key fingerprint ABF5BD827BD9BF62.
      如果输出下列字样，说明没问题
      pub rsa2048 2011-08-19 [SC] [expires: 2024-06-14]
      573B FD6B 3D8F BC64 1079 A6AB ABF5 BD82 7BD9 BF62
      uid [ unknown] nginx signing key
    5. 安装ngnix
      6. sudo apt update
      sudo apt install nginx
